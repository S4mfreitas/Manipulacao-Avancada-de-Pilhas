# Manipulacao-Avancada-de-Pilhas
Exercício de AEDS II (UNIMONTES) sobre o TAD Pilha. Implementação de funções para consulta de topo, concatenação de duas pilhas e criação de cópias (deep copy). O código respeita o encapsulamento, operando apenas via interface pública, utilizando recursividade ou pilhas auxiliares para manipulação dos dados.
